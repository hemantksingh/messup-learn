# Common shortcuts

|Function                                                           | Shortcut       |
|-------------------------------------------------------------------|:--------------:|
|Select language mode                                               | `Ctrl K + M`   |
|Format file                                                        | `Alt+Shift+F`  |
|Toggle word wrap                                                   | `Alt+Z`        |
|Remove line with criteria `^.*"key": "value",.*$\n` in json file   | `Repace All`   |
